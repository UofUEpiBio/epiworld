void ModelDiagram::read_transitions ( const std::string &  fn_transition)
inline

Reads the transitions from a file.

The file is assumed to come from the output of [Model::write_data()], and should be in the following format:

0 "S" "I" 10
1 "I" "R" 5
2 "R" "S" 3
...

The first column is the step, the second column is the state from which the transition is made, the third column is the state to which the transition is made, and the fourth column is the number of transitions.

Parameters
fn_transitionThe name of the file to read.
