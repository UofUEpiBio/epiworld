\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename TSeq = EPI\+\_\+\+DEFAULT\+\_\+\+TSEQ$>$\newline
class Model\+Measles\+Entity\+Quarantine$<$ TSeq $>$}

Measles model with entity-\/level quarantine. 

This class implements a Measles epidemiological model based on the SEIR framework with entity-\/level quarantine features including\+:
\begin{DoxyItemize}
\item Population mixing based on contact matrices
\item Measles-\/specific disease progression\+: Susceptible → Exposed → Prodromal → Rash
\item Prodromal individuals are infectious (replace the \char`\"{}\+Infected\char`\"{} state in SEIR)
\item Rash individuals are no longer infectious but can be detected for isolation
\item Entity-\/level quarantine measures (no contact tracing)
\item Quarantine duration varies based on vaccination prevalence within entities
\item Isolation policies for detected individuals during the rash state
\item Hospitalization of severe cases
\item Individual willingness to comply with public health measures
\end{DoxyItemize}

The model supports 13 distinct states\+:
\begin{DoxyItemize}
\item Susceptible\+: Individuals who can become infected
\item Exposed\+: Infected but not yet infectious (incubation period) ~\newline

\item Prodromal\+: Infectious individuals in the community (replaces \char`\"{}\+Infected\char`\"{} in SEIR)
\item Rash\+: Non-\/infectious individuals with visible symptoms (detection occurs here)
\item Isolated\+: Detected individuals in self-\/isolation
\item Isolated Recovered\+: Recovered individuals still in isolation
\item Detected Hospitalized\+: Hospitalized individuals who were detected
\item Quarantined Exposed\+: Exposed individuals in quarantine due to entity quarantine
\item Quarantined Susceptible\+: Susceptible individuals in quarantine due to entity quarantine
\item Quarantined Prodromal\+: Prodromal individuals in quarantine due to entity quarantine
\item Quarantined Recovered\+: Recovered individuals in quarantine due to entity quarantine
\item Hospitalized\+: Individuals requiring hospital care
\item Recovered\+: Individuals who have recovered and gained immunity
\end{DoxyItemize}
