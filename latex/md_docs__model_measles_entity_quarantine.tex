A comprehensive epidemiological model that implements measles transmission with entity-\/level quarantine measures, where quarantine duration varies based on vaccination prevalence within entities.\hypertarget{md_docs__model_measles_entity_quarantine_autotoc_md3}{}\doxysection{Overview}\label{md_docs__model_measles_entity_quarantine_autotoc_md3}
{\ttfamily \mbox{\hyperlink{class_model_measles_entity_quarantine}{Model\+Measles\+Entity\+Quarantine}}} extends the measles transmission framework with\+:


\begin{DoxyItemize}
\item {\bfseries{Measles-\/specific disease progression}}\+: Susceptible → Exposed → Prodromal → Rash → Recovered
\item {\bfseries{Population mixing}}\+: Uses contact matrices to model heterogeneous mixing between population groups
\item {\bfseries{Infectious period}}\+: Agents are infectious during the Prodromal state
\item {\bfseries{Detection and isolation}}\+: Detection occurs during the Rash state when symptoms become visible
\item {\bfseries{Entity-\/level quarantine}}\+: When a case is detected, quarantine is applied to all eligible individuals within the same entity (no contact tracing)
\item {\bfseries{Vaccination prevalence-\/based quarantine duration}}\+: Quarantine duration varies based on vaccination prevalence within each entity
\item {\bfseries{Isolation policies}}\+: Individual isolation for detected cases
\item {\bfseries{Hospitalization}}\+: Severe cases requiring hospital care
\item {\bfseries{Individual willingness}}\+: Agents have individual willingness to comply with public health measures
\end{DoxyItemize}\hypertarget{md_docs__model_measles_entity_quarantine_autotoc_md4}{}\doxysection{Key Differences from Model\+Measles\+Mixing}\label{md_docs__model_measles_entity_quarantine_autotoc_md4}

\hypertarget{md_docs__model_measles_entity_quarantine_autotoc_md14}{}\doxysection{Quarantine Duration Logic}\label{md_docs__model_measles_entity_quarantine_autotoc_md14}
The quarantine duration for each entity is determined dynamically based on vaccination prevalence\+:


\begin{DoxyEnumerate}
\item {\bfseries{Calculate vaccination prevalence}} for the entity (proportion of agents with vaccination tools)
\item {\bfseries{Find matching threshold}} in the {\ttfamily vaccination\+\_\+prevalence\+\_\+thresholds} vector
\item {\bfseries{Apply corresponding duration}} from the {\ttfamily quarantine\+\_\+durations} vector
\end{DoxyEnumerate}

For example, with thresholds {\ttfamily \mbox{[}0.\+0, 0.\+5, 0.\+8\mbox{]}} and durations {\ttfamily \mbox{[}21, 14, 7\mbox{]}}\+:
\begin{DoxyItemize}
\item \mbox{\hyperlink{class_entity}{Entity}} with 30\% vaccination → 21 days quarantine
\item \mbox{\hyperlink{class_entity}{Entity}} with 60\% vaccination → 14 days quarantine ~\newline

\item \mbox{\hyperlink{class_entity}{Entity}} with 90\% vaccination → 7 days quarantine
\end{DoxyItemize}\hypertarget{md_docs__model_measles_entity_quarantine_autotoc_md15}{}\doxysection{Entity-\/\+Level Quarantine Process}\label{md_docs__model_measles_entity_quarantine_autotoc_md15}
When a case is detected (agent reaches Rash state)\+:


\begin{DoxyEnumerate}
\item {\bfseries{\mbox{\hyperlink{class_entity}{Entity}} quarantine triggered}} for the entity containing the detected case
\item {\bfseries{All eligible agents}} in that entity are considered for quarantine\+:
\begin{DoxyItemize}
\item Must be in Susceptible, Exposed, Prodromal, or Rash state
\item Vaccinated agents (those with tools) are exempt
\item Subject to individual quarantine willingness
\end{DoxyItemize}
\item {\bfseries{Quarantine duration}} determined by current vaccination prevalence in the entity
\item {\bfseries{State transitions}} to appropriate quarantined states
