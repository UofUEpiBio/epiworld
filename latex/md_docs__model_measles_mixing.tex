A comprehensive epidemiological model that combines measles-\/specific disease progression with population mixing and quarantine measures.\hypertarget{md_docs__model_measles_mixing_autotoc_md3}{}\doxysection{Overview}\label{md_docs__model_measles_mixing_autotoc_md3}
{\ttfamily \mbox{\hyperlink{class_model_measles_mixing}{Model\+Measles\+Mixing}}} implements a measles transmission model that extends the SEIR framework with\+:


\begin{DoxyItemize}
\item {\bfseries{Measles-\/specific disease progression}}\+: Susceptible → Exposed → Prodromal → Rash → Recovered
\item {\bfseries{Population mixing}}\+: Uses contact matrices to model heterogeneous mixing between population groups
\item {\bfseries{Infectious period}}\+: Agents are infectious during the Prodromal state
\item {\bfseries{Detection and isolation}}\+: Detection occurs during the Rash state when symptoms become visible
\item {\bfseries{Contact tracing}}\+: Comprehensive contact tracing with configurable success rates
\item {\bfseries{Quarantine measures}}\+: Multiple quarantine states for exposed contacts
\item {\bfseries{Vaccination}}\+: Vaccine distribution with configurable efficacy and recovery enhancement
\end{DoxyItemize}\hypertarget{md_docs__model_measles_mixing_autotoc_md4}{}\doxysection{Disease States}\label{md_docs__model_measles_mixing_autotoc_md4}
The model includes 13 distinct states\+:


\begin{DoxyEnumerate}
\item {\bfseries{Susceptible}} -\/ Can become infected
\item {\bfseries{Exposed}} -\/ Infected but not yet infectious (incubation period) ~\newline

\item {\bfseries{Prodromal}} -\/ Infectious individuals (replaces \char`\"{}\+Infected\char`\"{} in standard SEIR)
\item {\bfseries{Rash}} -\/ Non-\/infectious with visible symptoms (detection occurs here)
\item {\bfseries{Isolated}} -\/ Detected individuals in self-\/isolation
\item {\bfseries{Isolated Recovered}} -\/ Recovered individuals still in isolation
\item {\bfseries{Detected Hospitalized}} -\/ Hospitalized individuals who were contact-\/traced
\item {\bfseries{Quarantined Exposed}} -\/ Exposed individuals in quarantine
\item {\bfseries{Quarantined Susceptible}} -\/ Susceptible individuals in quarantine
\item {\bfseries{Quarantined Prodromal}} -\/ Prodromal individuals in quarantine
\item {\bfseries{Quarantined Recovered}} -\/ Recovered individuals in quarantine
\item {\bfseries{Hospitalized}} -\/ Individuals requiring hospital care
\item {\bfseries{Recovered}} -\/ Immune individuals
