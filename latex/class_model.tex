\begin{Indent}\textbf{ Setting and accessing parameters from the model}\par
{\em \mbox{\hyperlink{class_tools}{Tools}} can incorporate parameters included in the model. Internally, parameters in the tool are stored as pointers to an std\+::map$<$$>$ of parameters in the model. Using the {\ttfamily epiworld\+\_\+fast\+\_\+uint} method directly fetches the parameters in the order these were added to the tool. Accessing parameters via the {\ttfamily std\+::string} method involves searching the parameter directly in the std\+::map$<$$>$ member of the model (so it is not recommended.)

The {\ttfamily par()} function members are aliases for {\ttfamily get\+\_\+param()}.

\DoxyCodeLine{[\mbox{\hyperlink{class_model_a343ab95f43273186d42d6df618695283}{name}} of parameter 1]: [value in double]}
\DoxyCodeLine{[\mbox{\hyperlink{class_model_a343ab95f43273186d42d6df618695283}{name}} of parameter 2]: [value in double]}
\DoxyCodeLine{...}

\end{DoxyCode}
