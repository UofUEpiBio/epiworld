Reads the transitions from a file. 

The file is assumed to come from the output of \mbox{[}\mbox{\hyperlink{classepiworld_1_1_model_a573869d1fe944d8a8345dc4a5bfc31bc}{Model\+::write\+\_\+data()}}\mbox{]}, and should be in the following format\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{0 \textcolor{stringliteral}{"{}S"{}} \textcolor{stringliteral}{"{}I"{}} 10}
\DoxyCodeLine{1 \textcolor{stringliteral}{"{}I"{}} \textcolor{stringliteral}{"{}R"{}} 5}
\DoxyCodeLine{2 \textcolor{stringliteral}{"{}R"{}} \textcolor{stringliteral}{"{}S"{}} 3}
\DoxyCodeLine{...}

\end{DoxyCode}
 The first column is the step, the second column is the state from which the transition is made, the third column is the state to which the transition is made, and the fourth column is the number of transitions.
