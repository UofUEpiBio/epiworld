\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename TSeq = EPI\+\_\+\+DEFAULT\+\_\+\+TSEQ$>$\newline
class Model\+Measles\+Mixing\+Risk\+Quarantine$<$ TSeq $>$}

Measles model with population mixing and risk-\/based quarantine strategies. 

This class extends the Measles epidemiological model to support different quarantine strategies based on exposure risk levels\+:


\begin{DoxyItemize}
\item {\bfseries{High Risk}}\+: Unvaccinated agents who share entity membership with the case
\item {\bfseries{Medium Risk}}\+: Unvaccinated agents who contacted an infected individual but don\textquotesingle{}t share entity membership ~\newline

\item {\bfseries{Low Risk}}\+: Other unvaccinated agents
\end{DoxyItemize}

Each risk level can have different quarantine durations, allowing for targeted public health interventions. The model also includes enhanced detection during active quarantine periods.

Disease progression follows the same states as \mbox{\hyperlink{class_model_measles_mixing}{Model\+Measles\+Mixing}}\+: Susceptible → Exposed → Prodromal → Rash → Recovered
